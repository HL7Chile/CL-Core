Resource Profile: Perfil Paciente para Receta Digital, extensible a Core Nacional

Defining URL:http://minsal.cl/fhir/core//StructureDefinition/CorePacienteCl
Version:0.8 (Ballot Pendiente)
Name:PacienteCl
Title:Perfil Paciente para Receta Digital, extensible a Core Nacional
Status:Active as of 2021-09-10T11:02:47-03:00
Definition:

Este Perfil ha sido desarrollado para cubrir las necesidades del Caso de Uso de Receta Electrónica. Sin embargo, se ha modelado con el fin de cubrir las necesidades nacionales de un Recurso Paciente para un Historial Clínico Nacional
